**Engineering-first web development agency** building stunning, scalable digital experiences.

## ✨ Features

- **Modern Stack**: React 18 + TailwindCSS + Framer Motion + CRACO
- **Stunning Animations**: 3D cubes, typing effects, glassmorphism, particle systems
- **Responsive Design**: Mobile-first, perfect on all devices
- **Full Stack Backend**: FastAPI/Python (optional)
- **Legal Pages**: Terms, Privacy with professional styling
- **Performance**: Optimized with Tailwind purge, CRACO config

## 🚀 Quick Start

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend (Optional)
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload
```
